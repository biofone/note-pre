## 数据结构
### 线性表
- 一个线性表是n个数据元素的优先序列。
- 线性表的顺序表示和链式表示
1. 线性表的线性表示指用一组连续的存储单元依次存储线性表的数据元素。
0. 线性表的链式存储结构的特点是用一组任意的存储单元存储线性表的数据元素（这组存储单元可以是连续的，也可以是不连续的）。  
   2.1 结点=数据域+指针域 ---> n个结点链结成一个链表，即为线性表的链式存储结构。  
   2.2 链表的每个结点中包含一个指针域，称为线性链表或单链表。  
   2.3 循环链表
   2.4 双向链表
